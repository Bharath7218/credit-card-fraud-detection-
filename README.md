# AustralianCredit
Fraud Detection in Credit Card Applications
Overview

This project focuses on detecting potential fraud in credit card applications using the Statlog (Australian Credit Approval) Dataset from UCI Machine Learning Repository
.

The workflow combines Self-Organizing Maps (SOMs) for unsupervised anomaly detection with a supervised Artificial Neural Network (ANN) to classify fraud probabilities.
